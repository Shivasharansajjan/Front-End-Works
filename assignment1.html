Mathematical Formulas

  1. (a+b)2 = a2 + b2 + 2ab
  2. (a-b)2 = a2 + b2 - 2ab
  3. a2 - b2 = (a+b)(a-b)
  4. (a+b)3 = a3 + b3 + 3ab(a+b)
  5. (a+b)4 = a4 + b4 + 4a3b + 6a2b2 + 4ab3
  6. sin2(θ) + cos2(θ) = 1
  7. ex = ∑n=0∞ (xn / n!)
  8. loga(bx) = loga(b) + loga(x)
  9. ax = ex log(a)
  10. (a+b)n = ∑k=0n (C(n, k) an-k bk) (Binomial Theorem)

Chemical Reactions

  1. Combustion of Methane: CH4 + 2O2 → CO2 + 2H2O
  2. Formation of Water: 2H2 + O2 → 2H2O
  3. Synthesis Reaction: N2 + 3H2 → 2NH3
  4. Decomposition of Water: 2H2O → 2H2 + O2
  5. Neutralization Reaction: HCl + NaOH → NaCl + H2O
  6. Photosynthesis: 6CO2 + 6H2O → C6H12O6 + 6O2
  7. Rusting of Iron: 4Fe + 3O2 + 6H2O → 4Fe(OH)3
  8. Combustion of Glucose: C6H12O6 + 6O2 → 6CO2 + 6H2O
  9. Acid-Base Reaction: H2SO4 + 2NaOH → Na2SO4 + 2H2O
  10. Formation of Bicarbonate: CO2 + H2O → H2CO3 (Carbonic Acid)
